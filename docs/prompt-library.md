# Prompt Library (v0)

Reusable snippets for vibe-coding conversations. Copy-paste into the chat panel to scaffold games quickly.

## Scene seeds
- **Neon runner:**
  > "Create a side-scrolling runner in a neon city. Player can jump + dash. Add parallax skyline and collectible energy cores."
- **Cosmic garden:**
  > "Top-down garden with 5 planter tiles. Player waters seedlings, plants grow over 30 seconds."
- **Boss room:**
  > "Single arena, add circular boundary, spawn boss that fires projectiles every 2 seconds."

## Mechanics
- **Pickup system:**
  > "Add pickup entities tagged `battery`. When player overlaps, increase score by 10, play `powerup.wav`, destroy pickup."
- **Dialogue overlay:**
  > "Overlay text box bottom of screen. Provide function `showDialogue(lines: string[])` that cycles on key press."
- **Touch controls:**
  > "Enable touch joystick left (move) and button right (jump)."

## Art direction modifiers
- "Use flat pastel sprites with thick outlines"
- "CRT shader effect"
- "Paper-cutout aesthetic with drop shadows"

## Export prompts
- Web build: "Export current game to static web build"
- Desktop: "Prepare desktop package named 'Neon Runner' for macOS + Windows"
- Mobile: "Generate Capacitor project with portrait lock and touch controls"
